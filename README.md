# TensorFlow Beginner's Toolkit

A complete guide and working codebase for beginners to learn TensorFlow.

## Quick Start

1. Clone repo: `git clone https://github.com/shara-arch/tensorFlow-Beginner-Toolkit.git`
2. Create virtual env: `python -m venv tf-env`
3. Activate: `source tf-env/bin/activate` (Mac/Linux) or `tf-env\Scripts\activate` (Windows)
4. Install deps: `pip install -r requirements.txt`
5. Verify setup: `python setup_verification.py`
6. Run examples: `python examples/01_hello_tensorflow.py`
7. Open notebooks: `jupyter notebook notebooks/`

## What's Inside

- **TOOLKIT_GUIDE.md** - Complete 8-section learning guide
- **examples/** - Runnable Python scripts with explanations
- **notebooks/** - Interactive Jupyter notebooks
- **setup_verification.py** - Test your installation

See TOOLKIT_GUIDE.md for full documentation!
